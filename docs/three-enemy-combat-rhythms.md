# 3体限定の戦闘リズム実験

対象は鉄の処刑人、古竜、共鳴クリスタルだけ。他敵へは適用しない。

## 鉄の処刑人

装甲中は直接ダメージ75%減。通常攻撃では装甲を壊せない。防御で受け流し準備を行い、次の処刑を防御すると装甲が崩れ、次のプレイヤー1行動は敵へのダメージ+100%。既存heavy counterplayの反撃態勢+30%はこの敵では付与せず、火力20%中断またはCC中断を防御禁止時の代替崩壊手段として共有する。

## 古竜

飛翔は2プレイヤー行動続き、直接ダメージ70%減。攻撃しても飛翔は解除されない。2行動後にブレスへ移り、ブレス実行後は2プレイヤー行動だけ過熱して被ダメージ+60%。飛翔中は防御・回復・バフ・状態異常・CD待ちを優先し、過熱中に攻撃スキルを集中する。

## 共鳴クリスタル

通常攻撃、攻撃スキル、防御/構え、状態異常、回復をカテゴリとして記録する。直前と同じカテゴリの効果は80%減。異なる3カテゴリで障壁が崩れ、次の1プレイヤー行動は被ダメージ+50%。契約でカテゴリが不足する場合の詰み防止として、同カテゴリ反復でも4回目に障壁は崩れるが、それ以前の反復攻撃は軽減される。HPが減っていない時に回復薬を使うことはBotの優先手段にしない。

## UI

敵カードに現在状態、次状態/残りプレイヤー行動数、直接ダメージ倍率、有効カテゴリを表示する。弱点中は金色に強調する。各行動ボタンは有効なら通常表示、軽減対象なら半透明表示とし、操作自体は禁止しない。

## 計測上の注意

Bot指標は軽減中への直接攻撃、露出中攻撃、露出前のスキル温存、防御崩し、飛翔中準備、過熱中スキル、結晶カテゴリ使用/反復、敵別行動分布を記録する。初回結果だけを報告し、数値の反復調整は行わない。
